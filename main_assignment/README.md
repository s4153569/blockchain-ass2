# Secure DLT-Based Inventory Management System

This folder contains the Task 1 and Task 2 implementation for INTE2627 Blockchain Technology Fundamentals Assignment 2.

The system demonstrates secure record insertion in a distributed inventory environment using:

- RSA digital signatures
- SHA-256 hashing
- Signature verification
- Simplified permissioned majority consensus
- Local record storage across inventory nodes
- Harn-style multi-signature approval
- RSA encrypted response delivery
