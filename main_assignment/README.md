# Secure DLT-Based Inventory Management System

This folder contains the integrated Task 1, Task 2, and Task 3 implementation for INTE2627 Blockchain Technology Fundamentals Assignment 2.

The system demonstrates secure record insertion and secure query retrieval in a distributed inventory environment using:

- RSA digital signatures
- SHA-256 hashing
- Signature verification
- Simplified permissioned majority consensus
- Local record storage across inventory nodes
- Harn-style multi-signature approval
- RSA encrypted response delivery
